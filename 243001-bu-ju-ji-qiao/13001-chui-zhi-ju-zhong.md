利用 flex 实现垂直居中



```
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8" />
		<title></title>
		<style>
			div{
				height: 100px;
				background: brown;
				display: flex;
			}
			
			p{
				margin: auto;
			}
		</style>
	</head>
	<body>
		
		<div>
			<p>我是测试文字</p>
		</div>
		
	</body>
</html>

```
